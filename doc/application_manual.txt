1 . From the editor (Text Editor)
Double-click a word to highlight all matching words.
Double-click the tab area to display the path in the explorer tree or use the keyboard shortcut [Ctrl + Shift + Q].
Move the mouse to the tab area and use the keyboard shortcut [Ctrl + Shift + C] to copy the full current path.
Use the keyboard shortcut [Ctrl + Shift + E] to open the window explorer tab at the parent folder.
Use the keyboard shortcut [Ctrl + O] to open the file.
Use the keyboard shortcut [Ctrl + Shift + O] to open a folder.
Use the keyboard shortcut [Ctrl + N] to create a new editor tab.
Use the keyboard shortcut [Ctrl + S] to save the current changes.
Use the keyboard shortcut [Ctrl + Z] to undo changes on the editor.
Use the keyboard shortcut [Ctrl + Y] to redo the changes on the editor.
Use keyboard shortcut [Ctrl + F] to open quick search popup.
Use the keyboard shortcut [Ctrl + H] to open a quick alternative pop-up.
Use the keyboard shortcut [Ctrl + G] to open the Goto dialog box.
Use the keyboard shortcut [Ctrl + Shift + G] to quickly search all selected text.
Use the keyboard shortcut [Ctrl + Shift + N] to open a new VinaText instance.
Use the keyboard shortcut [Ctrl + Shift + T] to open a recently closed file.
Use the keyboard shortcut [Ctrl + Alt + T] to open a terminal tab at the container folder.
Use the keyboard shortcut [Ctrl + Shift + W] to open the opened file manager dialog.
Use the keyboard shortcut [Ctrl + W] to close the current file.
Use the keyboard shortcut [F4] to reload the file contents.
Use keyboard shortcut [F3] to search selected text on google browser.
Use the keyboard shortcut [F1] to translate the selected text into the preset language.
Use keyboard shortcut [Ctrl + Shift + M] to fold/unfold all brackets.
Use the keyboard shortcut [Ctrl + Shift + V] to paste the text into the quote.
Use the keyboard shortcut [Windows + V] to open the clipboard history dialog box.
Use the keyboard shortcut [Ctrl + J] to disable editing mode.
Use the keyboard shortcut [Ctrl + D] to expand the selection to the next matching word.
Use the keyboard shortcut [Ctrl + Shift + A] to expand the selection to all matching words, to a single quote.
Use the keyboard shortcut [Alt + L] to split the selection into lines.
Use the keyboard shortcut [Alt + Left Mouse Drag] to use column editing mode.
Use the keyboard shortcut [Ctrl + Alt + DownArrow] to extend the selection to the next line.
Use the keyboard shortcut [Ctrl + Alt + UpArrow] to extend the selection down to the previous line.
Use the keyboard shortcut [F5] to run the current file.
Use the keyboard shortcut [Shift + F5] to stop the current program.

2 . From File Explorer (File Explorer Tree)
Use the keyboard shortcut [F5] to refresh the tree.
Use the keyboard shortcut [Ctrl + Enter] to open the current path by default application.
Use the keyboard shortcut [Ctrl + Shift + C] to copy the current path in full.
Use the keyboard shortcut [Ctrl + Alt + T] to open a terminal tab at the current path.
Use the keyboard shortcut [Ctrl + Shift + Q] to bring focus back to the editor.
Use the keyboard shortcut [Ctrl+F] to open the Search Paths/Navigation dialog box.
Use the keyboard shortcut [Ctrl + Shift + F] to search for text in the current directory.
Use the keyboard shortcut [Ctrl + Shift + H] to replace the text at the current directory.
Use keyboard shortcut [Ctrl + G] to jump to any path.
Use the keyboard shortcut [Ctrl + Shift + P] to open the file template dialog box.
Use the keyboard shortcut [Ctrl + Shift + M] to collapse the current directory.
Use the keyboard shortcut [Ctrl + Shift + N] to expand the current directory.
Use the keyboard shortcut [Ctrl + M] to create a new folder.
Use the keyboard shortcut [Ctrl + N] to create a new file.
Use the [Delete] shortcut to send the file to the trash.
Use the keyboard shortcut [Ctrl + B] to highlight a path.
Use the [Tab] shortcut to display only one path.
Use the keyboard shortcut [Shift + Tab] to go back to the root folder on the desktop.
Use the keyboard shortcut [F2] to rename the path.
Use the keyboard shortcut [Shift + Right-click] to open the system menu.
